# Phase 2 產品設計 — 月統計頁籤

## 概述

在現有運營看板新增「月統計」頁籤，顯示 8 項核心指標的月度彙總與環比變化。

## 資料模型

### 後端彙總 (`aggregate.py`)

在 `aggregate()` 內，於 `rev`、`cp` 字典建成後（sets 尚未轉為 counts），執行月度彙總：

```
月度分組: YYYY-MM = day[:7]
每組彙總:
  投注總額  = Σ(rev 每日投注)
  有效打碼  = Σ(rev 每日有效打碼)
  淨利 NGR  = Σ(rev 每日 NGR)
  充值總額  = Σ(cp 每日充值)
  提現總額  = Σ(cp 每日提現)
  活躍會員  = |∪(rev 每日 active 會員 set)|   ← 跨日去重
  新增註冊  = Σ(reg_by_day 每日新增)
  首充會員  = Σ(fc_by_day 每日首充)
```

環比 = (本月 / 上月 − 1) × 100%（首月或上月為 0 時標註「—」）

## UI 設計

### 導航列

現有單一「運營看板」頁籤 + 新增「月統計」頁籤：

```
[ 运营看板 | 月统计 ]
```

點擊切換顯示對應區塊（`document.getElementById('board')` vs 新 `'monthly_board'`）。

### 月統計頁面

以月份為行、指標為列的橫向可捲動表格：

| 月份 | 投注总额 | NGR | 有效打码 | 充值 | 提现 | 活跃 | 新注册 | 首充 |
|------|----------|-----|----------|------|------|------|--------|------|
| 2026-07 ⏳ | 567k ▲12% | -50k ▼8% | ... | ... | ... | 234 ▲5% | 56 ▼11% | 12 — |
| 2026-06 | 1.23M — | -50k — | ... | ... | ... | 222 | 63 | 12 |

- 最新月份在最上面（倒序）
- 當月標示「⏳ 進行中」
- 每格：數值 + 環比指示（▲/▼ 百分比，綠色升/紅色降）
- 首月無環比顯示「—」

### 手機適配

表格 `overflow-x: auto` 可橫向捲動。第一欄（月份）`position: sticky; left: 0` 固定。
