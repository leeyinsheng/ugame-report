# v1.14 — 週統計 / 月統計新增到帳彩金與實際返水

## 背景

當前週統計與月統計的 NGR 已扣除彩金與返水，但運營團隊無法直接從報表中看到這兩項扣除的具體金額。後端資料已存在（`m["彩金"]`、`m["返水"]`），僅需補上輸出。

## 目標

在週統計與月統計表格中新增「到帳彩金」與「實際返水」欄位。

## 範圍

### 包含
- `_monthly_stats()` / `_weekly_stats()` 輸出 entry 新增到帳彩金、實際返水
- 兩項指標納入環比計算
- 前端 `MONTHLY_COLS` 新增對應欄位
- Unit test 更新

### 不包含
- 每日看板異動（彩金與返水已存在）
- 其他區塊變動

## 規格

| 項目 | 到帳彩金 | 實際返水 |
|------|---------|---------|
| 資料來源 | `m["彩金"]`（ledger 彩金類） | `m["返水"]`（ledger 返水類） |
| 表頭 | 「到帳彩金」 | 「實際返水」 |
| 位置 | GGR 之後、實際返水之前 | 到帳彩金之後、淨利潤 NGR 之前 |
| 環比 | ✅ | ✅ |
| 格式 | 金額類，小數點後 2 位 | 金額類，小數點後 2 位 |
