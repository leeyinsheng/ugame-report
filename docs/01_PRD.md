# Phase 1 產品需求文檔 — 週統計頁籤

## 需求背景

運營看板已有每日與月度數據，需補充**週度**視角，便於追蹤短期趨勢與週對比。

## 功能範圍

### 後端

| 項目 | 規格 |
|------|------|
| 函數 | `_weekly_stats(days, rev, cp, reg_by_day, fc_by_day, today=None)` |
| 週定義 | ISO 週一至週日，以 `date.isocalendar()[1]` 取週號，年+週號組 key |
| 指標 | 8 項（與月統計相同） |
| 彙總規則 | 金額 sum、活躍會員 union（跨日去重） |
| 環比 | vs 上週（本週/上週 − 1 × 100%） |
| 進行中 | 本週標註，由 `today` 參數控制（預設當日） |
| 輸出位置 | `_meta.weekly` 陣列，每筆含 `週次`, `日期段`, 8 指標, `前週環比%`, `進行中` |

### 前端

| 項目 | 規格 |
|------|------|
| 導航 | 新增「週統計」頁籤（第 3 個 tab，位於運營看板與月統計之間） |
| 表格 | 橫向可捲動，週次 + 日期段 sticky，8 指標 + 環比（▲/▼） |
| 進行中標籤 | 本週顯示「⏳ 進行中」、已結週顯示「✓」 |
| 順序 | 最新週在上，倒排 |

## 資料格式範例

```json
{
  "_meta": {
    "weekly": [
      {
        "週次": "2026-W28",
        "日期段": "07/06 → 07/12",
        "投注总额": 567890.12,
        "净利润NGR": -50234.56,
        "有效打码": 423456.78,
        "充值总额": 234567.89,
        "提现总额": 67890.12,
        "活跃会员": 341,
        "新增注册": 89,
        "首充会员": 23,
        "前週环比%": {
          "投注总额": 12.5,
          "净利润NGR": -8.2,
          ...
        },
        "进行中": true
      }
    ]
  }
}
```

## 驗收標準

- [ ] 導航列顯示三個頁籤：运营看板 | 週統計 | 月統計
- [ ] 週統計表格顯示 8 項指標，數值格式正確
- [ ] 本週標註「進行中」，已結週無標註
- [ ] 環比正確（vs 上週，首週無環比）
- [ ] 活躍會員跨日去重
- [ ] 不影響既有每日/月度功能
