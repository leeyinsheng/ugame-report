# Feature Verification — 活動彩金名稱映射 v1.3

## 驗證項目

### 1. API 輸出格式

後端 `/api/summary` 的 `bonus` 欄位結構保持不變：

```json
{
  "as_of": "2026-07-03",
  "summary": {
    "累计到帐彩金": 1768.88,
    "当日新增到帐彩金": 15.53,
    "领取会员": 74,
    "整体兑现率": 37.44
  },
  "list": [
    { "活动": "一般充值", "到帐彩金": 1422.79, ... },
    { "活动": "首次充值", "到帐彩金": 250.0, ... },
    { "活动": "负盈利", ... },
    ...
  ]
}
```

前端 `renderBonus()` 直接迭代 `b.list`，每個 item 的 key 名稱不變 → **無需前端修改** ✅
